# bower # 
> Também é um gerenciador de pacotes como o npm porém focado no frontend

Inspirado no [vídeo](https://www.youtube.com/watch?v=R7CWXUwaKVA) do [Rafael Dias](https://github.com/eudiasrafael)

## Instalação ##
* `npm install -g bower`

## Iniciando ##
* `bower unit`
